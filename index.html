Privacy Policy for WFT Helper Chrome Web Extension

**Effective Date:** September 29, 2025

This Privacy Policy describes how the **WFT Helper Chrome Web Extension** ("the Extension") handles user data. The Extension is designed to synchronize your work schedule from The Home Depot's proprietary Workforce Tools portal to your personal Google Calendar.

The core principle of this Extension's design is local data processing and user privacy.

1. Data Collection and Processing

The WFT Helper Extension operates entirely within your local web browser environment. **We do not collect, transmit, store, or sell any user data on our own external servers.** All processing and storage of sensitive information are performed locally on your device.

A. Data Processed Locally

The Extension processes the following data types directly within your browser:

Data Type Source Purpose of Processing
**Work Schedule Details** The Home Depot Workforce Tools portal (`https://wft.homedepot.com/`) via `webRequest` interception. Used solely to format shifts (date, time, location) into Google Calendar events.
**Workforce User Data (Name)** The Home Depot Workforce Tools API endpoints. Used solely to personalize the name of the dedicated Google Calendar (e.g., "John Doe's WFT Calendar").

B. Data Stored Locally (using `chrome.storage`)

The Extension uses `chrome.storage.sync` and `chrome.storage.local` to store the following configuration data, ensuring persistent, local functionality:

  • **Google Access Token:** A temporary token used to authorize access to the Google Calendar API.
  • **Google Calendar ID:** The unique identifier for the dedicated WFT Helper Calendar created in your Google account.
  • **Google User Email:** Stored for validation purposes, ensuring the user is syncing to the calendar created with the correct Google account.
  • **Extension State:** Includes timestamps (e.g., last refresh) and Home Depot internal API URLs (endpoints captured via `webRequest`).

C. Data Shared with Third Parties (Google Calendar API)

The following data is shared with the Google Calendar API (a third party), initiated only by the user:

  • **Work Shift Events:** Date, start time, end time, and shift location are sent to the Google Calendar API to create or update events in the user's personal calendar.

2. Permissions Justification

The Extension requests the following permissions to fulfill its sole function of synchronizing your schedule:

  • **Host Permission (`https://wft.homedepot.com/`):** Required to inject the content script and use `webRequest` to capture internal schedule data endpoints.
  • **`webRequest`:** Required for monitoring network traffic exclusively on the Home Depot domain to securely obtain the raw schedule data.
  • **`identity`:** Required to initiate the secure Google OAuth flow, allowing the user to grant access to the Google Calendar API.
  • **`storage`:** Required for local, secure storage of configuration data and authentication tokens.
  • **`activeTab`:** Required to ensure the synchronization process is initiated successfully upon user interaction (clicking the extension icon).

3. Children's Privacy

The WFT Helper Extension is not intended for use by individuals under the age of 18. We do not knowingly process or store data from children.

4. Changes to This Policy

We may update our Privacy Policy from time to time. We will notify users of any changes by updating the effective date of this policy. Continuing to use the Extension after changes are made signifies your acceptance of the revised policy.
